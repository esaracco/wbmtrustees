Affichage des ACLs

Affiche la liste de toutes les ACLs déjà configurées pour Trustees

  • En cliquant sur "Ajouter", vous pouvez ajouter un nouveau chemin (répertoire ou fichier) sur lequel gérer des droits spécifiques. Il est possibe de saisir manuellement le chemin ou bien de sélectionner un chemin en parcourant l'arborescence du serveur. Pour ce faire, cliquez sur le bouton "..." situé à gauche du bouton "Ajouter". Une nouvelle fenêtre s'ouvre, à partir de laquelle vous pourrez parcourir l'arborescence. Une fois votre choix effectué, cliquez sur le bouton "Ok". Le chemin sélectionné s'affiche dans le champ texte.

  • Pour gérer les droits sur les chemins cliquez sur celui de votre choix. Vous entrez alors en mode édition (cf. l'aide de la page d'édition).

  • Vous pouvez supprimer une règle en cliquant sur le bouton "Supprimer" dans le panel situé à droite de chaque ligne. La ligne sera définitivement supprimée du fichier de configuration Linux Trustees. Si vous désirez juste désactiver la régle sans la supprimer du fichier, cliquez sur le bouton "Désactiver". Après un rechargement de la page le lien correspondant sera inactif et la couleur de fond indiquera son nouvel état. Pour réactiver une ligne désactivée, cliquez sur le bouton "Activer". Le lien deviendra actif et la couleur de fond de la ligne redeviendra normale.
